# VENDIXPRO - Punto de Venta Profesional

Esta aplicación es un sistema de Punto de Venta (POS) avanzado construido con Next.js y Firebase.

## Características
- **Punto de Venta (POS)**: Gestión de ventas con múltiples métodos de pago (Efectivo, Tarjeta, QR, Cuenta Corriente).
- **Inventario**: Control de stock y precios en tiempo real.
- **Clientes**: Gestión de deudas y cuentas corrientes.
- **Gastos**: Registro de egresos operativos.
- **Caja**: Control de apertura y cierre de caja diario.
