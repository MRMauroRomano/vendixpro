# **App Name**: VENDIXPRO

## Core Features:

- Punto de Venta (POS): Permitir la selección de productos, la gestión de un carrito de compra, el cálculo de totales, impuestos y la generación de tickets de venta con opciones de pago.
- Gestión de Stock e Inventario: Funcionalidades para listar, agregar, editar y eliminar productos, incluyendo detalles como cantidad, precio, categoría y proveedor. Generación de alertas automáticas para stock bajo.
- Historial de Ventas: Registro completo de todas las ventas realizadas con opciones de filtrado por fecha, producto o usuario. Funcionalidad para exportar el historial de ventas a CSV o PDF.
- Control de Caja Diario: Funcionalidad para la apertura y cierre diario de caja, registrando ingresos, egresos y ajustes. Mostrar un resumen claro del saldo disponible en tiempo real.
- Página de Gastos: Permitir registrar gastos operativos y categorizarlos. Vincularlos con el control de caja.
- Dashboard de Rendimiento: Proporciona una visión general con las ventas del día, semana y mes, destacando productos con stock bajo y gráficos visuales de ingresos y egresos.
- Gestión de Cuentas Corrientes: Registrar clientes con crédito ('fiado'), gestionar pagos parciales, calcular la deuda total y ver el historial completo de transacciones por cliente.
- Autenticación de Usuarios: Sistema de autenticación robusto para gestionar el acceso de usuarios, garantizando seguridad y permitiendo un control diferencial sobre las funcionalidades del sistema.
- Integración con Firebase: Utilizar Firebase Studio para el backend y base de datos en tiempo real o Firestore, guardando todas las operaciones (ventas, stock, caja, gastos, clientes) en Firebase.

## Style Guidelines:

- Esquema de color primario 'Verde Oscuro' (#1B4D3E) para una estética profesional y estable.
- Fondo 'Blanco' (#FFFFFF) para una interfaz limpia, clara y de alta legibilidad.
- Color de acento 'Verde Vibrante' (#38DB51) utilizado para elementos interactivos y destacados, ofreciendo un contraste moderno.
- La tipografía principal será 'Poppins' (sans-serif), elegida por su diseño moderno y alta legibilidad, ideal para interfaces con mucha información como listados y formularios.
- Se utilizará un conjunto de iconos modernos y minimalistas, con un estilo que evoque la simplicidad y eficiencia. Para el logo principal y elementos clave de compra, se incluirá un icono distintivo de 'bolsita de compra'.
- Diseño completamente responsive y adaptable a cualquier tamaño de pantalla (celulares y notebooks), priorizando una estructura clara y un flujo de información intuitivo.
- Animaciones sutiles y fluidas para transiciones entre vistas, retroalimentación al usuario (por ejemplo, confirmaciones de guardado) y microinteracciones, mejorando la experiencia sin distraer.
